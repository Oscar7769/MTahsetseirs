\chapter{Introduction and Motivation}
\label{ch:intro}

近年來，人工智慧(AI)與雲端數據中心的快速發展，使得世界各地對高效能運算(High-Performance Computing, HPC)的算力需求大幅提升。
然而，傳統數據中心與晶片內部的電互連(Electrical Interconnects)架構受限於金屬導線歐姆損耗造成的焦耳熱(Joule heating)、
信號衰減以及頻寬瓶頸，已面臨嚴重的能源損耗與傳輸極限。
為了突破此物理限制，矽光子(Silicon Photonics)技術因具備高頻寬、
低延遲及低功耗等優勢，被視為未來能夠取代傳統電子傳輸、實現高效能晶片資訊傳遞的關鍵技術。
同時，隨著矽光子元件架構走向高密度整合與大規模光子積體電路(Photonic Integrated Circuits, PICs)發展，
其設計參數與幾何自由度也大幅增加，使得複雜光學結構的反向設計成為一項具挑戰性的研究課題。

在矽光積體電路中，為了追求更高效能並實現更密集的光路排列，透過縮小晶片尺寸來提升整合度已經成為未來趨勢，
這使得緊湊型(Compact)光學元件的重要性也日益增加。在複雜的晶片佈局中，光訊號傳輸無法只依靠直線波導，
還必須透過彎曲波導(Bend Waveguide)來引導光路轉彎，並利用交叉波導(Waveguide Crossing)實現不同光路之間的交會，
以提升晶片佈局效率。此外，隨著模式分波多工(Mode-Division Multiplexer, MDM)技術逐漸成熟，
模態分波解多工器(Mode-Division Demultiplexer, DEMUX)也成為多模態矽光子系統中的關鍵元件，負責將不同傳播模態有效分離至對應輸出通道。
然而，當元件尺寸縮小至微米甚至奈米尺度時，彎曲波導容易因光場偏移而產生額外損耗;
交叉波導則是面臨著模態散射所導致的插入損耗與通道間的信號串擾(Crosstalk);
而模態分波解多工器則必須在有限的設計空間內同時兼顧高模態轉換效率、低串擾與高模態純度，使其設計難度大幅提升。
因此如何在有限尺寸下設計出具有低損耗、高傳輸效率以及高模態光場純度的矽光子元件，成為目前矽光子晶片反向設計的瓶頸。

目前，在學術界與產業界設計矽光子元件時，主要依賴兩大類方法，第一種方法是基於傳統光學經驗與解析模型的\textbf{啟發式試誤法}(Heuristic
 trial-and-error Approach)。雖然此方法可以有效設計出標準光學元件，但在面對較為複雜的光學結構時，其性能往往受限且難以預測。
第二種方法則是使用當前最主流的\textbf{基於梯度的伴隨變分法}(Adjoint Method)來進行光學結構的拓撲優化(Topology Optimization)。
此方法只需要進行一次正向電磁模擬(Forward Simulation)與一次伴隨模擬即可計算出梯度，
並在每次迭代中根據梯度來更新結構。然而，這種基於梯度的優化機制本質上非常容易陷入局部最佳解(Local Minima)，
且優化過程中容易產生許多細小、破碎的分支與孤島結構。這種結構不僅對製程誤差極度敏感，
更常常因為結構尺寸小於最小製程極限而導致其無法通過微影與蝕刻進行量產，形成了設計與實際製造之間的巨大落差。

近年來，研究人員曾提出因子分解機(Factorization Machine, FM)結合基於退火演算法之方法，並成功應用於光學薄膜與熱發射器等結構設計問題。
受到此研究的啟發，本研究進一步將 FM模型 與模擬退火結合，建立一套光學結構反向設計架構，
以提升高維設計空間中的全域搜尋能力，並降低傳統梯度式最佳化容易陷入局部最佳解的問題。

然而，相較於結構較為單純的薄膜元件，二維矽光子元件對幾何邊界的連續性與局部細節特徵具有更高的要求。
若直接採用傳統像素化(Pixelated)二元編碼，不僅容易產生階梯效應(Staircase Effect)，造成額外散射損耗，也難以兼顧不同元件所需的幾何結構特徵。
例如，彎曲波導與交叉波導著重於平滑且連續的導波路徑，而模態分波解多工器則需同時形成大尺度導波結構與局部細小結構，以實現不同模態之有效分離，
因此對結構表示方式提出更高的要求。
因此本研究導入三種編碼策略，包括高斯上採樣(Gaussian Upsampling)、二維傅立葉積分(2D Fourier Integral)與 $\beta$-變分自編碼器($\beta$-Variational Autoencoder, $\beta$-VAE)，
藉由連續與低維化的結構表示方式，來降低設計空間複雜度，同時提升幾何結構的表示能力與代理模型的學習效率。
透過上述編碼策略與 FMSA 架構之結合，本研究建立了一套兼具全域搜尋能力與製造可行性的矽光子反向設計架構。

為了驗證所提出方法之有效性，本研究分別針對三種矽光子元件進行反向設計，
包括 90 度彎曲波導、交叉波導以及模態分波解多工器。
本論文之主要貢獻如下：
\begin{itemize}
\item \textbf{提出多種編碼策略並結合全域優化架構:}本研究首次將 Gaussian Upsampling 與 2D Fourier Integral 作為二維矽光子結構之編碼策略，
並將原先應用於週期性結構生成之 $\beta$-VAE 延伸至對邊界連續性具有嚴格要求之矽光子波導元件設計。

\item \textbf{完成高效能且具製造可行性之矽光子元件設計:}我們成功設計出低損耗、高傳輸效率之 90 度彎曲波導，
以及具備低插入損耗與低串擾特性之交叉波導與 DEMUX 結構，同時維持平滑邊界與良好的製造可行性，其中交叉波導在維持緊湊元件尺寸下，
傳輸效率與串擾表現皆優於相關文獻，驗證此研究所提出的反向設計流程具有良好的最佳化能力與製造可行性。

\item \textbf{分析不同編碼策略於矽光子元件之適用範圍:}我們探討Gaussian Upsampling、2D Fourier Integral及$\beta$-VAE三種編碼策略應用於90度彎曲波導、
交叉波導及模態分波解多工器等不同元件的優化表現，分析各方法於幾何連續性、設計自由度、多尺度結構及代理模型學習能力等方面之特性，
建立不同結構表示方法於各類矽光子元件中的適用範圍。

\item \textbf{建立適用於不同矽光子元件之品質因子與設計流程:}我們依據不同元件的設計需求建立不同的品質因子，
並結合編碼策略作為結構表示方法與建立最佳化流程，形成適用於彎曲波導、交叉波導及模態分波解多工器等元件反向設計方法，
可作為未來光子積體電路(Photonic Integrated Circuits, PIC)反向設計與多模態光學元件開發之參考。

\end{itemize}
